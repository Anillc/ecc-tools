## Failure log

- First build attempt failed because `~STAAdapter() override` was inherited from `ResettableInterface` whose destructor is non-virtual (`protected: ~ResettableInterface() = default;`). Removed the `override` keyword (kept the `~STAAdapter()` declaration moved to .cc so the unique_ptr<incomplete> compiles). Build recovered.
- No other rollbacks were required. The W2b namespace migration compiled in one shot because all unqualified `clock_trace::` calls inside `ClockTraceResolver.cc` were resolvable from within the new `icts::sdc` namespace once the helper free functions were also in `icts::sdc`, so they just lost the namespace prefix (same enclosing namespace).
