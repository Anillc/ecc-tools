### Naming changes summary

- 5 chapter `.cc` slices (`AnalyticalSolverXxx.cc`) → 5 stage-named `.cc` files reflecting their BEAM-search pipeline role.
- Each stage gets a companion `.hh` facade that re-exports the W1b declarations relevant to its stage, with a doxygen comment describing the stage role.
- Public API of `AnalyticalSolver` unchanged; `SolveAnalyticalHTreeCandidates` retains identical signature.
- All public function declarations remain in W1b headers (unmodified); new stage `.hh` files include those W1b headers to make stage boundaries discoverable.
- All anonymous-namespace helpers stay anonymous, just migrated alongside their public callers to the new stage `.cc`.
